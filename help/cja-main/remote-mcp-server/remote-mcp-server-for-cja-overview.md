# Présentation du serveur MCP distant pour [!DNL Customer Journey Analytics]

Découvrez comment connecter des outils d’IA tels que [!DNL Claude], [!DNL ChatGPT] ou [!DNL Cursor] au serveur MCP d’Adobe Customer Journey Analytics (CJA) pour explorer les vues de données, exécuter des rapports et générer des informations sur les indicateurs de performance clés prêts à l’emploi à l’aide du langage naturel. Cette démonstration montre comment les serveurs MCP distants activent des workflows d’analyse évolutifs et régis, ainsi que des rapports automatisés dans Adobe Experience Platform.

Pour plus d’informations, consultez la [documentation](https://developer.adobe.com/analytics-mcp/docs/cja/){target="_blank"} de Content Analytics.
