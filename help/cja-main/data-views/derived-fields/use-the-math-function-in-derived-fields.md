# Ajuster les mesures à l’aide de la fonction mathématique dans les champs dérivés de Customer Journey Analytics

Dans ce tutoriel vidéo, découvrez le potentiel de l’utilisation des fonctions mathématiques pour créer des champs dérivés dans Customer Journey Analytics. Découvrez comment ajuster les mesures en supprimant des totaux les composants indésirables tels que les taxes et comprenez les avantages de l’utilisation de champs dérivés par rapport aux mesures calculées, y compris les fonctionnalités de vue de données exclusives telles que la déduplication, le regroupement et l’attribution. Bénéficiez d’une plus grande flexibilité pour créer des mesures personnalisées qui correspondent à vos besoins en matière d’analyse, améliorant vos capacités de prise de décision axée sur les données.
