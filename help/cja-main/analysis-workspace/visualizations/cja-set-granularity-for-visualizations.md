# Définir la granularité des visualisations

Ajustez facilement l’affichage de vos données à l’aide du sélecteur de granularité dans les paramètres de visualisation des tendances. Cette fonctionnalité simplifie l’analyse des données en vous permettant d’affiner les visualisations en fonction de vos besoins. Découvrez comment optimiser vos workflows et rendre les informations sur les données plus accessibles.
