# Entonnoir

Découvrez comment **&#x200B;**&#x200B;dans Customer Journey Analytics fournit une représentation visuelle d’un parcours utilisateur critique dans votre produit et vous aide à déterminer les points de friction.

>[!VIDEO](https://video.tv.adobe.com/v/3421663/?learn=on)

Vous pouvez également comparer plusieurs événements en une seule étape funnel, créant ainsi un « funnel dupliqué ». Ce « funnel dupliqué » vous permet de comparer la friction de deux parcours côte à côte, ce qui peut s’avérer utile lorsqu’il existe des options d’étape ou qu’une expérience A/B est en cours d’exécution dans le funnel.

>[!VIDEO](https://video.tv.adobe.com/v/3431113/?learn=on)

Pour plus d’informations, consultez la [documentation](https://experienceleague.adobe.com/fr/docs/analytics-platform/using/guided-analysis/funnel/friction){target="_blank"}.
