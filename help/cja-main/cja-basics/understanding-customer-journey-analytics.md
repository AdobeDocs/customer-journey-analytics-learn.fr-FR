# Comprendre Customer Journey Analytics

Adobe Customer Journey Analytics (CJA) est un service d’application reposant sur Adobe Experience Platform. Il intègre Analysis Workspace à Platform, un outil d’analyse riche qui vous permet d’effectuer des analyses intercanaux sur n’importe quel jeu de données de Platform.

>[!VIDEO](https://video.tv.adobe.com/v/30090/?quality=12&enable10seconds=on&speedcontrol=on)

Pour plus d’informations, consultez la [documentation](https://docs.adobe.com/content/help/fr-FR/analytics-platform/using/cja-landing.html).
